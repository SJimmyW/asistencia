# asiste
**Open-source attendance and formative assessment system for higher education.**
---

## Philosophy

The application is **never** the center of the system.
The center of the system is **Google Sheets**.
The application simply reads and writes information in Google Sheets and provides a simple interface for teachers and students.
A teacher should be able to manage an entire course **without editing a single line of R code**.

---
## Main features
- Attendance using QR codes.
- Formative assessment after attendance.
- Automatic correction of multiple-choice questions.
- Teacher dashboards.
- Course evolution.
- Attendance tracking by student.
- Google Sheets as the only database.
- Mobile-friendly interface.
- Open source.

---

## Project principles
